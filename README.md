# LLP Web

Public student website for the Lecture Learning Platform.

## Stack

- Next.js 16 App Router
- React 19
- TypeScript
- Tailwind CSS 4
- ESLint
- Prettier

## Scripts

```bash
npm run dev
npm run build
npm run lint
npm run lint:fix
npm run typecheck
npm run format
npm run format:check
npm run check
```

## Structure

Application code lives in `src/`.

See `docs/project-structure.md` for the working folder convention.
